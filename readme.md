User Stories:

- Als gebruiker wil ik toegang hebben tot een geïntegreerde versie van Google Maps die zich beperkt tot de gemeente Groningen. 
- Als gebruiker wil ik alle gemeentelijke monumenten op de kaart weergegeven zien.
- Als gebruiker wil ik op individuele gemeentelijke monumenten kunnen klikken en de bijgeleverde informatie op kunnen vragen.
